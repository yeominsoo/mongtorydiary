# Mongtory Diary 단일 세션 단계적 업무계획

## 목적
Spring Boot 백엔드와 Flutter 앱을 중심으로 실제 사용 가능한 감정형 일기 MVP를 완성하기 위한 실행 계획이다. 2026-05-21부터 역할별 분배는 철회됐으므로, 남은 작업은 하나의 세션이 순서대로 처리한다.

## 현재 기준선
- 백엔드: 회원가입, 로그인, 토큰 재발급, 일기 CRUD, 캘린더, 감정 목록, 오늘/위젯 요약 API가 있다.
- Flutter: 로그인 입력 폼, 일기 목록/상세/생성/수정/삭제, 캘린더 날짜 탭 진입, 작성 UX 개선, mock/remote 전환 구조가 있다.
- QA 이력: widget test 하네스, remote CRUD API 검증, 오류 계약 검증, 캘린더 날짜 탭 회귀 검증 기록이 있다.
- 협업 운영: PM/QA/FE/BE 역할 분배는 철회됐고, 단일 세션 작업 목록은 `.ai-work/msyeo/docs/single-session-worklist.md`에서 관리한다.
