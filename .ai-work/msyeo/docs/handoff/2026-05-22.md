## 미해결 이슈
- 현재 작성 리마인더는 앱 내부 설정 상태다. Android/iOS OS 알림 예약은 네이티브 권한, 스케줄링, 저장 경계가 필요하다.
- PostgreSQL은 개발 편의를 위해 `ddl-auto=update`를 사용 중이다. 운영 전 Flyway 또는 Liquibase 마이그레이션 전환이 필요하다.
- 업로드 파일은 로컬 파일 시스템 저장이다. 운영 전 영구 볼륨, 백업, 삭제 정책이 필요하다.
- `/run/mongtorydiary-db.env`는 재부팅 후 사라질 수 있다. 정식 systemd unit 또는 k3s/Kubernetes 구성에서는 secret/env 관리를 분리해야 한다.
- 서비스는 systemd 임시 unit이다. 재부팅 후 자동 실행이 필요하면 정식 unit 또는 Kubernetes 배포로 전환한다.
- 기존 미추적 항목 `.codex`, `node_modules/`, `package-lock.json`, `package.json`는 이번 작업과 무관해 커밋 대상에서 제외한다.
