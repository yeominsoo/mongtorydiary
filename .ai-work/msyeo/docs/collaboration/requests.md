# 역할 간 요청 인덱스

## 사용법
- 이 문서는 PM이 관리하는 공통 요청 인덱스다.
- 각 역할 세션은 새 요청을 자기 역할 디렉토리의 `requests.md`에 먼저 작성한다.
- PM은 역할별 요청 문서를 확인한 뒤 이 인덱스와 대상 역할 `inbox.md`에 반영한다.
- 세션 간 동시 편집을 줄이기 위해 QA, FE, BE 세션은 이 파일을 직접 수정하지 않는 것을 원칙으로 한다.
- 상세한 논의와 긴 메모는 역할별 문서에 남기고, 이 문서에는 추적에 필요한 요약만 둔다.
