- 로그인 성공 후 access token 발급을 확인했다.

- 감정 목록 조회를 확인했다.
- invalid access token 일기 목록 요청은 401 `ApiResponse` 형태로 내려오는 것을 확인했다.
- Flutter `test` 7건과 `analyze`는 통과했다.
- `REQ-20260428-06`은 QA 역할 문서 기준 완료 상태다.
- 요청자 PM의 `responses.md`에도 `RES-REQ-20260428-06`을 기록했다.
