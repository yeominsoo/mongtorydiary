# FE 요청 문서

## 목적
FE 개발자가 PM, QA, BE 개발자에게 요청하거나 이슈를 전달할 때 사용하는 문서다.

## 요청 목록
| 요청 ID | 우선순위 | 대상 역할 | 상태 | 제목 | 생성일 |
| --- | --- | --- | --- | --- | --- |
| 없음 | - | - | - | 현재 FE가 발행한 요청 없음 | - |

## 작성 템플릿
```text
### REQ-YYYYMMDD-NN
- 우선순위:
- 요청자: FE
- 대상 역할:
- 상태: 대기
- 요청 내용:
- 완료 기준:
- 관련 파일/영역:
```
